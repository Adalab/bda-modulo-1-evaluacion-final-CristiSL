# EVALUACIÓN FINAL MÓDULO 1 - CRIS SAENZ 🚀

Este es mi proyecto para la evaluación final del Módulo 1 del curso de Data Analytics & IA de Adalab. En este módulo aprendemos Python desde 0 y en esta evaluación se nos proponen una serie de ejercicios, algunos más simples y otros más complejos, para trabajar con todo lo aprendido y ponernos a prueba.

# TECNOLOGÍAS

Python, Jupyter Notebook, Markdown.

# ESTRUCTURA DEL PROYECTO

La estructura es muy simple. Los títulos y enunciados de los ejercicios se presentan en Markdown para que se vea más limpio. Debajo del enunciado viene el código que resuelve cada ejercicio y cada línea de código tiene un comentario explicativo de lo que hace. Al final de los 7 ejercicios obligatorios, inician los ejercicios del Bonus.

# CÓMO EJECUTAR EL PROYECTO

Para ejecutar el proyecto desde GitHub, sigue estos pasos:

1. Clona el repositorio en tu equipo:
   
   `git clone https://github.com/Adalab/bda-modulo-1-evaluacion-final-CristiSL.git`
   
2. Abre la carpeta del proyecto en VS Code (o la herramienta que tú uses).
3. Abre el archivo **evaluacion_final_modulo1.ipynb**.
4. Si trabajas en VS Code, asegúrate de tener instalada la extensión de Jupyter.
5. Ejecuta las celdas del notebook o usa `Run All`.
6. Revisa los resultados en las celdas de salida y prueba las funciones interactivas.
